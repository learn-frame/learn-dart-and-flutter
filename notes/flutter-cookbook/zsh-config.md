# 关于 配置 zsh 的 Flutter 环境

```bash
# Flutter
export PUB_HOSTED_URL=https://pub.flutter-io.cn
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn
export PATH=~/code/flutter/bin:\$PATH # ~/code/flutter 是你 flutter 的路径
export FLUTTER_ROOT=~/code/flutter # 必须配置这一条, 否则无法使用 dartdoc, dartanalyzer 等命令行工具
```
